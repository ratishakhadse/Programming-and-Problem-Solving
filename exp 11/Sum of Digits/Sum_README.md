
## Problem Statement
Write a Python program that reads an integer, calculates the sum of its digits, and prints the result.

---

## Algorithm
1.Start

2.Read number num

3.If num == 0 → return 0

4.Else → return last digit + sum of remaining digits

5.Call function

6.Print result

7.Stop

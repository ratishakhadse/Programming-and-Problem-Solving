
## Problem Statement
Write a program that takes a string as input and prints the total count of vowels in that string. Consider both uppercase and lowercase.

---

## Algorithm
1.Start

2.Read string st

3.Set count = 0

4.Traverse each character of string

5.If character is vowel → count = count + 1

6.Print count

7.Stop

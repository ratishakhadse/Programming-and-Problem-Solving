
## Problem Statement
Write a Python program that prints the reverse of a given input string. The program uses a reverse_string() function that takes a string as input and returns its reversed string.

---

## Algorithm
1.Start

2.Define function to reverse string

3.Initialize empty string

4.Traverse string in reverse order

5.Append characters to new string

6.Return reversed string

7.Input string

8.Call function

9.Print original and reversed string

10.Stop


## Problem Statement
Write a Python program to remove all punctuation marks from a given string and print the resulting string.

---

## Algorithm
1.Start

2.Read string st

3.Set new_str = ""

4.Traverse each character

5.If character is alphanumeric or space → add to new_str

6.Print new_str

7.Stop
